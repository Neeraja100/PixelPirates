# The Financial Mirror

### Clarity is Luxury.

**The Financial Mirror** is a premium, data-driven introspection tool designed to elevate your financial consciousness. By transforming raw transaction data into a "deep space" visual reflection, it helps you see your money, understand your habits, and refine your lifestyle.

![Hero Preview](https://stitch.withgoogle.com/static/media/reflective-monolith-hero.png) *Visual style inspired by the Stitch "Reflective Monolith" design system.*

---

## ✨ Features

- **Ambient Design**: A sleek, dark-mode interface featuring glassmorphism, glowing gradients, and premium typography (Manrope & Inter).
- **Introspection Engine**: An AI-powered backend that analyzes your transactions to generate personality profiles, SWOT assessments, and actionable financial nudges.
- **Privacy First**: Localized processing and one-click data deletion. Your financial reflection belongs only to you.
- **Dynamic Dashboard**:
  - **Health Score**: A gradient-based visual of your financial well-being.
  - **Action Engine**: Numbered, priority-graded steps to improve your habits.
  - **Visual Insight**: Interactive charts with custom glass tooltips.

---

## 🚀 Tech Stack

- **Frontend**: React, Tailwind CSS, Recharts, Framer Motion (animations).
- **Backend**: FastAPI, Python, OpenAI/Gemini integration.
- **Design System**: Stitch "Reflective Monolith" (Deep Space Neutrals, Violet/Purple accents).

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- Python 3.10+
- OpenAI API Key (configured in backend)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Neeraja100/PixelPirates.git
   cd PixelPirates
   ```

2. **Setup Backend**
   ```bash
   cd backend
   pip install -r requirements.txt
   # Setup your environment variables in a .env file
   python main.py
   ```

3. **Setup Frontend**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```

## 🛡️ Security Architecture

We prioritize the safety of your financial data through:
- **Private Session Initialization**: No permanent storage of raw statements.
- **End-to-End Encryption**: Secure parsing and analysis protocols.
- **Privacy Protocol**: Built-in adherence to localized data processing principles.
